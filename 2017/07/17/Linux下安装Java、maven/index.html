maven依赖Java，所以安装maven需要先安装Java。

1. 安装Java

首先去Oracle官网去下载Java的压缩包，找到Linux x64那一行点击jdk-8u131-linux-x64.tar.gz会从该下载链接（类似http://download.oracle.com/otn-pub/java/jdk/8u131-b11/d54c1d3a095b4ff2b6607d096fa80163/jdk-8u131-linux-x64.tar.gz?AuthParam=1500262277_e257bad3c9bf43d1c34178b3630a8fe1 ）进行下载，拷贝该链接，用命令行进行下载。

1.1 下载Java压缩包

在linux命令行下，执行下载命令：

wget http://download.oracle.com/otn-pub/java/jdk/8u131-b11/d54c1d3a095b4ff2b6607d096fa80163/jdk-8u131-linux-x64.tar.gz?AuthParam=1500262277_e257bad3c9bf43d1c34178b3630a8fe1

1.2 解压压缩包

等一会就下载好了Java压缩包 jdk-8u131-linux-x64.tar.gz
然后解压：

tar -zxvf jdk-8u131-linux-x64.tar.gz

1.3 设置Java环境变量：

vim /etc/profile

在该文件末尾加上（我的压缩包是解压在/usr/local/目录下）：

export JAVA_HOME=/usr/local/jdk1.8.0_131
export CLASSPATH=.:$JAVA_HOME/lib
export PATH=$JAVA_HOME/bin:$PATH

1.4 使配置文件立即生效

source /etc/profile

1.5 查看是否安装成功：

java -version

2. 安装maven

首先去maven官网（http://maven.apache.org/download.cgi ），找到apache-maven-3.5.0-bin.tar.gz 的下载链接（类似http://mirrors.tuna.tsinghua.edu.cn/apache/maven/maven-3/3.5.0/binaries/apache-maven-3.5.0-bin.tar.gz ）。

2.1 下载maven压缩包

wget http://mirrors.tuna.tsinghua.edu.cn/apache/maven/maven-3/3.5.0/binaries/apache-maven-3.5.0-bin.tar.gz

2.2 解压

tar apache-maven-3.5.0-bin.tar.gz

2.3 配置maven环境变量：

vim /etc/profile

在该文件末尾加上（我的压缩包是解压在/usr/local/目录下）：

export MAVEN_HOME=/usr/local/apache-maven-3.5.0
export PATH=:$MAVEN_HOME/bin:$PATH

2.4 使配置文件立即生效

source /etc/profile

2.5 查看是否安装成功：

mvn -v
